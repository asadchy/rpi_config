Ядро:
1. git clone https://github.com/asadchy/linux.git
2. git clone https://github.com/asadchy/rpi_config.git
3. файл .config из директории rpi_config скопировать в директорию linux
4. перейти в директорию linux
5. компилируем ядро, модули, dtb: make -j8 ARCH=arm CROSS_COMPILE=arm-linux-gnueabihf- zImage modules dtbs
6. после окончания компиляции, подключить sd карточку
7. все последующие команды выполняются из директории linux
8. sudo cp arch/arm/boot/zImage <path_to_mount_point>/boot/kernel7.img
9. sudo cp arch/arm/boot/dts/*.dtb <path_to_mount_point>/boot/
10. sudo cp arch/arm/boot/dts/overlays/*.dtb* <path_to_mount_point>/boot/overlays/
11. sudo make ARCH=arm CROSS_COMPILE=arm-linux-gnueabihf- INSTALL_MOD_PATH=<path_to_mount_point>/rootfs/ modules_install
